FEATURES: 

1. A user-friendly web UI with following features:
	- Text inputs or buttons to filter data.
	- Options or buttons to sort the data.
	- A results list or table to show the data.

2. If one of the characters in the table is selected, a detailed image appears below the table

3. Filtering and sorting operations

4. Adjustable page size (how many results to display)

5. Error management

##########################################################################

NOTES FROM DEVELOPER:
- Both filtering and sorting cannot be done at the same time. When applied separately,
these operations will be listed in the table successfully.

- To run the script, navigate to the project file path in the terminal.
Run “python app.py” and click http://127.0.0.1:5000. Web UI will open in your default browser.

- The processes may be a bit slow, please wait.
